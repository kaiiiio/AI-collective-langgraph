# Slide 23: LangSmith Mental Model

## Concept
Shift from control flow to observability. LangSmith turns each graph run into a trace with nested node, model, and tool runs.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask what trace evidence you would inspect after a bad output.
