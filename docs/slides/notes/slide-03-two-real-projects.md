# Slide 3: Two Real Projects

## Concept
Introduce the two case studies. The content workflow teaches scale, retries, and schema validation. The licensing workflow teaches evidence, review, and auditability.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask students what both workflows have in common even though the domains are different.
