# Slide 24: LangSmith Tracing Code

## Concept
Show traceable wrappers around important functions. Good names make traces readable for both engineers and reviewers.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask what names would be useful for the content and licensing projects.
