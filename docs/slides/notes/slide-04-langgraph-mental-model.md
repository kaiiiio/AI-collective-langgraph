# Slide 4: LangGraph Mental Model

## Concept
Explain the graph loop slowly: state enters a node, the node returns a partial update, and edges choose the next node. Keep returning to the question: what changed in state?

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask students to point to state, node, and edge in the text diagram.
