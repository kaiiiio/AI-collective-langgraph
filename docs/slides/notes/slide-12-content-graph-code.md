# Slide 12: Content Graph Code

## Concept
Show how the content nodes are registered. Students should notice that node names become visible labels in traces and debugging.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask why descriptive node names matter in LangSmith.
