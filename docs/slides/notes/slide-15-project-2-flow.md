# Slide 15: Project 2 Flow

## Concept
Introduce the licensing workflow as a regulated process. It needs evidence, ordered steps, audit trails, and human judgment more than it needs one clever prompt.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask which steps should never be fully automatic.
