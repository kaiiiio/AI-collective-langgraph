# Slide 21: Licensing Conditional Edges

## Concept
Show approval gates as routes. Human review is a node in the graph, not a vague meeting outside the workflow.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask students to trace a low-confidence classification path.
