# Slide 7: Project 1 Flow

## Concept
Treat the AI content system as an existing workflow that already wants to be a graph. The model call is one node; context loading, parsing, repair, validation, saving, and progress updates are separate responsibilities.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask which step should own retry behavior.
