# Slide 17: Licensing Nodes

## Concept
Show that each AI responsibility is bounded. Classification, mapping, and gap analysis are separate because they have different inputs, outputs, and risks.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask why combining all three into one prompt would be harder to review.
