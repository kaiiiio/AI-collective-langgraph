# Slide 11: Conditional Retry Code

## Concept
Explain conditional edges as normal Python decisions over state. Invalid JSON and retry_count determine whether the graph repairs, fails, or continues.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask what route happens when validation_errors exists and retry_count is 2.
