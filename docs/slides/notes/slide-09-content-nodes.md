# Slide 9: Content Nodes

## Concept
Use the node list to show bounded responsibility. A node should be easy to name because it does one job.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask what each node adds to state.
