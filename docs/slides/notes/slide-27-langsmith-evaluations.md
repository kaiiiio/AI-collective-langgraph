# Slide 27: LangSmith Evaluations

## Concept
Explain evaluations as repeatable quality checks. They convert subjective review into measurable signals such as schema match, location relevance, no fake claims, and grounded citations.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask students to propose one evaluation for the licensing workflow.
