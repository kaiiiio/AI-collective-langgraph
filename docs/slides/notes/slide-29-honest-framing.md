# Slide 29: Honest Framing

## Concept
Be explicit: these are real project flows that LangGraph can formalize. Do not claim the existing projects already use LangGraph unless that is true.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask students why honest framing matters in engineering demos.
