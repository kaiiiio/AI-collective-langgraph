# Slide 25: LangSmith Setup

## Concept
Explain the Docker-friendly setup: tracing config comes from .env and Compose passes it into containers. Do not hardcode API keys in graph code.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask why environment variables are safer than putting keys in source files.
