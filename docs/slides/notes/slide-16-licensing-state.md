# Slide 16: Licensing State

## Concept
Explain the state contract for a licensing case. It contains profile data, documents, chunks, classification, policies, rule mappings, gap findings, review decisions, and exports.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
Ask which fields represent evidence and which represent decisions.
