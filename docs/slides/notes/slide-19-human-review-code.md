# Slide 19: Human Review Code

## Concept
Explain that human review is not outside the system. The graph records what needs review and which gaps are high risk.

## How to Explain It
Use the inductive rhythm: name the concept, connect it to the real project example, point at the code or flow, then ask students to identify the state change.

## Student Check
