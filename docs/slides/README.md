# LangGraph Learning Lab Slides

This folder contains the public HTML slide deck.

Open:

```text
docs/slides/index.html
```

The deck is plain HTML, CSS, and JavaScript, so it can be opened directly in a browser or served by Vercel through the root rewrite in `vercel.json`.

## Instructor Files

Use these while teaching:

```text
docs/slides/speaker-script.md
docs/slides/notes/
docs/slides/outline.md
```

Keep presenter guidance in those Markdown files, not on the visible slides.
