# 60-Minute Teaching Plan

| Time | Activity | Outcome |
|---:|---|---|
| 0-5 min | AI agents overview | Students know why workflows need structure. |
| 5-15 min | LangGraph basics | Students can explain graph, node, edge. |
| 15-25 min | State + nodes + edges | Students can inspect state before and after execution. |
| 25-35 min | Tools + memory | Students can call deterministic tools and store history. |
| 35-45 min | Multi-agent | Students can describe role-based graph nodes. |
| 45-55 min | LangSmith | Students can explain traces, spans, latency, token usage, and debugging. |
| 55-60 min | Capstone demo | Students see all pieces combined. |

Use `./lab module <number>` or `.\lab.cmd module <number>` for live demos. Keep the Docker Compose details out of the
main lesson until the Docker module, so students focus on graphs, state, nodes, and edges first.

See [module-by-module-teaching-guide.md](module-by-module-teaching-guide.md) for the inductive teaching notes.
