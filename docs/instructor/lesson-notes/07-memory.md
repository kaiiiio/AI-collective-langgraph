# Memory

Memory lets an agent use prior context. Short-term memory is usually conversation history; persistent memory survives runs.

```mermaid
graph TD
    A["Concept"] --> B["Code"]
    B --> C["Example"]

```

## Instructor Notes

Start with the mental model, draw the graph, run the smallest possible example, then ask students to change
one thing. The repetition is intentional: concept, code, example, exercise, solution.
