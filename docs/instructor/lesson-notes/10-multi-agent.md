# Multi-Agent Systems

Multi-agent systems split work into specialized roles such as researcher, writer, and reviewer.

```mermaid
graph TD
    A["Concept"] --> B["Code"]
    B --> C["Example"]
    C --> D["Exercise"]
    D --> E["Solution"]
```

## Instructor Notes

Start with the mental model, draw the graph, run the smallest possible example, then ask students to change
one thing. The repetition is intentional: concept, code, example, exercise, solution.
