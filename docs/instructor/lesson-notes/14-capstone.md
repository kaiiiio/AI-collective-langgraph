# Capstone

The capstone combines routing, tools, memory, review, and LangSmith tracing into an AI research assistant.

```mermaid
graph TD
    A["Concept"] --> B["Code"]
    B --> C["Example"]
    C --> D["Exercise"]
    D --> E["Solution"]
```

## Instructor Notes

Start with the mental model, draw the graph, run the smallest possible example, then ask students to change
one thing. The repetition is intentional: concept, code, example, exercise, solution.
