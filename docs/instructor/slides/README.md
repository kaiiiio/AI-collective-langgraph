# LangGraph Learning Lab Slides

This folder contains a static HTML slide deck for teaching the LangGraph Learning Lab.

## Open Locally

Open `docs/instructor/slides/index.html` in a browser.

## Present

Use keyboard navigation:

- Right arrow or space: next slide
- Left arrow: previous slide
- Home: first slide
- End: final slide
- O: overview
- Escape: close overview

## Vercel

The root `vercel.json` rewrites `/` to this deck, so the repository can be deployed as a static site.
